Este    es  un  
    ejemplo     de  texto   preformateado

La marca prepreserva espacios en blanco y saltos de linea

La marca pre es buena para mostrar codigo de programacion

        public static void main(String args[])
    {
        System.out.println("Hola Mundo");
    }
}

Tambien podemos usar la marca code

public static void main(String args[]) { System.out.println("Hola Mundo"); } }

Pero la marca code no respeta el espaciado ni el salto de linea
